# Portfolio Website

Site portfolio personnel développé en HTML, CSS et JavaScript, conçu pour présenter mes projets, expériences et réalisations en data, business intelligence et analyse.

## Objectif
Créer une vitrine claire, moderne et professionnelle permettant de mettre en avant :
- mes projets data et BI
- mes expériences académiques et professionnelles
- mes compétences techniques
- mes liens de contact

## Contenu du site
- **Section d’introduction** avec présentation générale
- **Grille de projets et expériences** avec cartes interactives
- **Fenêtres modales** pour afficher une description détaillée et un slider d’images
- **Section contact** avec liens vers LinkedIn, GitHub, email et WhatsApp

## Technologies utilisées
- **HTML5** pour la structure
- **CSS3** pour le design et la mise en page
- **JavaScript** pour les interactions dynamiques (modales, slider, animations)

## Fonctionnalités principales
- Design responsive
- Cartes de projets cliquables
- Affichage détaillé de chaque projet dans une popup
- Slider d’images pour illustrer les réalisations
- Navigation fluide entre les sections

## Aperçu

Le site a été pensé comme un support de présentation professionnel pour accompagner mes candidatures et centraliser mes projets sur une seule interface.
